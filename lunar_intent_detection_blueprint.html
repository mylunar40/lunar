3. New Firestore Fields Required

Recommended: Extend users/{uid} Document

Addition to users/{uid}:

userJourney: {
  intent: "breakup_recovery" | "anxiety" | ...
  intentSelectedAt: timestamp
  cycleTracking: boolean
  lastMenstrualPeriod: date (nullable)
  cycleLengthDays: integer (nullable)
  personalizationEnabled: boolean
  personalizationConsentAt: timestamp
}

onboardingProgress: {
  completedIntentSelection: boolean
  completedCycleData: boolean
  completedPersonalizationConsent: boolean
  onboardingCompletedAt: timestamp
}

Why This Structure?

  • No extra document read (all user data in one place)
  • Supports all 7 intents
  • Optional cycle data (only populated if relevant)
  • Tracks onboarding completion

5. Onboarding Flow Specification

Path A: Brand New User (First Time)

[Auth] → [User Created] → [Check Intent]
Intent NULL? YES → [Show Intent Selection]
[Save Intent + Load Screen 2 Logic]
[Show Cycle Data Screen] (if relevant)
[Show Personalization Consent]
[Confirmation Card]
[Navigate to Home] → [Load Personalized Dashboard]

Path B: Returning User (App Reopened)

[App Launched] → [Auth Check] → [User Intent Exists?]
YES → [Skip Onboarding] → [Load Home with User Intent Loaded]

Screen Validation Rules

Screen Field Rule Error Message
Intent Selection intent Must select 1 of 7 "Please select your reason for joining"
Cycle Data LMP Valid date required if shown "Select a valid date"
Cycle Data Cycle length 15-45 days range "Typical cycle is 21-35 days"
Personalization — At least 1 checkbox "Enable at least one personalization"

6. Post-Selection Behavior (By Intent)

On User Selects Intent, Execute:

Breakup Recovery

1. Save intent = "breakup_recovery"
2. Skip cycle screen
3. Load home with: "Healing Moment" card
4. Auto-add "Breakup Healing" community circle
5. Fire analytics: "intent_selected:breakup_recovery"

Anxiety & Stress

1. Save intent = "anxiety"
2. Skip cycle screen
3. Load home with: "Breathe with Lunar" card
4. Auto-add "Anxiety Warriors" circle
5. Fire analytics: "intent_selected:anxiety"

Pregnancy

1. Save intent = "pregnancy"
2. SHOW cycle screen (required)
3. Calculate gestational week from LMP
4. Auto-add "Expecting Moms" + "Pregnancy Support" circles
5. Activate Pregnancy Mode

Cycle Wellness

1. Save intent = "cycle_wellness"
2. SHOW cycle screen (required)
3. Calculate current phase from LMP
4. Auto-add "Cycle Warriors" circle
5. Load home with: "Cycle Phase Insight" card

(Similar patterns for Self Love, Relationship, Emotional Growth)
